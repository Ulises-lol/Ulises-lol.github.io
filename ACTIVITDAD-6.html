Unidad 1 | Actividad 06 | Protocolos de Red

Lista de Protocolos

  1. OSI - Open Systems Interconnection (Interconexión de Sistemas Abiertos)

    Modelo conceptual que estandariza las funciones de comunicación de un sistema de telecomunicaciones.

  2. IP - Internet Protocol (Protocolo de Internet)

    Encargado de direccionar y enrutar los paquetes de datos a través de redes.

  3. TCP - Transmission Control Protocol (Protocolo de Control de Transmisión)

    Garantiza la entrega confiable de datos dividiéndolos en segmentos y reensamblándolos en el destino.

  4. UDP - User Datagram Protocol (Protocolo de Datagramas de Usuario)

    Protocolo de comunicación sin conexión, rápido pero sin garantía de entrega.

  5. SSH - Secure Shell (Intérprete de Órdenes Seguro)

    Protocolo para acceder y administrar dispositivos de forma segura a través de una red.

  6. FTP - File Transfer Protocol (Protocolo de Transferencia de Archivos)

    Permite la transferencia de archivos entre computadoras en una red.

  7. SMTP - Simple Mail Transfer Protocol (Protocolo de Transferencia de Correo Simple)

    Se usa para el envío de correos electrónicos entre servidores.

  8. POP - Post Office Protocol (Protocolo de Oficina de Correo)

    Permite la descarga de correos electrónicos desde un servidor al cliente.

  9. DHCP - Dynamic Host Configuration Protocol (Protocolo de Configuración Dinámica de Host)

    Asigna dinámicamente direcciones IP a dispositivos en una red.

  10. DNS - Domain Name System (Sistema de Nombres de Dominio)

    Convierte nombres de dominio en direcciones IP para facilitar el acceso a sitios web.

  11. HTTP - HyperText Transfer Protocol (Protocolo de Transferencia de Hipertexto)

    Protocolo usado para la comunicación entre navegadores y servidores web.

  12. ARP - Address Resolution Protocol (Protocolo de Resolución de Direcciones)

    Traduce direcciones IP a direcciones MAC en una red local.

  13. MAC - Media Access Control (Control de Acceso al Medio)

    Identificador único asignado a tarjetas de red para su comunicación en la red.

  14. LAN - Local Area Network (Red de Área Local)

    Red de computadoras que conecta dispositivos en un área geográfica limitada.
